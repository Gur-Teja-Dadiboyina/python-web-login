## ğŸ§  Project Overview

This project demonstrates a complete **DevOps automation pipeline** integrating:
- **Jenkins** for continuous integration and deployment  
- **Docker** for containerization  
- **Kubernetes (EKS)** for orchestration  
- **AWS CLI** for cluster management  
- **GitHub** as the SCM (Source Code Management) system  

Once a code change is committed to GitHub, Jenkins automatically:
1. Builds the Docker image of the application  
2. Pushes it to Docker Hub  
3. Updates the Kubernetes manifests with the new image tag  
4. Deploys the updated app to **AWS EKS**  
5. Retrieves and displays the **public URL** of the live app
